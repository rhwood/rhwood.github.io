Amazon: WTF?

Jan 16th, 2008

I’m going to be moving soon, so I signed up to Amazon’s EasySell service so as to be able to simply send them a box of old books and other media and not have to think about it.

When I login to the seller account, I get the following notice:

Seller Central works best on the Macintosh with Netscape version 7.1 or higher. If you encounter problems using this suite of tools we encourage you to use Netscape version 7.1 or higher.

The version of Netscape recommended was released in July 2003 and the Netscape browsers are no longer developed. What is Amazon doing supporting an older version of a discontinued product?
